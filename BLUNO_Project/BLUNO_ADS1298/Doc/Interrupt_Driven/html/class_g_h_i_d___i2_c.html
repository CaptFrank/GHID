Detailed Description

This class is a wrapper class for the I2C buss class already included in the Arduino framework. This class extends the TwoWire class, and still allows access to those functions.

See also:
Wire.h

Note: To start a transmission, the user must use the following sequence of events:

  • Init I2C,
  • Start transmission (address)
  • Send
  • Stop transmission (address)
