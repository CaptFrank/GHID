Detailed Description

This class is a class in which we can define two types of UART interfaces. Either software serial drivers can be instantiated or hardware serial bus drivers can be instantiated. This class extends the SoftwareSerial class and the HardwareSerial class which are both included in the Arduino IDE framework.
