byte  interrupt_number 
)

The trigger settings setter method.

This method sets a trigger on the digital port. The 2 parameters trigger value is the actual triggered value. This value can be on of the following:

  • LOW -> 0 (0V or reference voltage)

HIGH -> 1 (5V)

  • CHANGE -> 1 (When the pin toggles)
  • FALLING -> 2 (HIGH to LOW)
  • RISING -> 3 (LOW to HIGH)

While the function pointer is a void* to the function that will be activated when we get an interrupt on the digital interface.

Note: The interrupts can only be set on pins {2, 3, 21, 20, 19, 18}. These are mapped to {0, 1, 2, 3, 4, 5} respectively.

Parameters:
trigger_value- byte triggered value
function_ptr- void* function that will be activated

The write port method.

value- byte either 1 or 0

  • 01010001 & 0x01 = 1
  • 00101000 & 0x01 = 0
