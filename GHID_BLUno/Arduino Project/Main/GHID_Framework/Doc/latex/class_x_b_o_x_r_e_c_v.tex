\subsection{\-Detailed \-Description}
\-This class implements support for a \-Xbox \-Wireless receiver.

\-Up to four controllers can connect to one receiver, if more is needed one can use a second receiver via the \hyperlink{class_u_s_b_hub}{\-U\-S\-B\-Hub} class.
